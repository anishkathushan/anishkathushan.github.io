Recent Projects

WS Ray - Teleoperated Underwater Drone for Immersive Reef Inspection

This project aims to provide an immersive way for users to remotely explore shallow coral reefs through a tethered underwater drone. Featuring a CFD-optimized hull for stable maneuverability, sensor-fusion navigation for safe path planning, and real-time video streaming with mounted lighting, the system reduces diver presence and vessel traffic while safeguarding fragile reef habitats. It supports ecological research, public engagement, and sustainable marine tourism in Sri Lanka.

MEMS Based Microgripper for Biological Cell Manipulation

Designed and simulated a MEMS microgripper with force-feedback for safe, precise manipulation of biological cells (10 - 100 μm), improving handling reliability in biomedical research.

Smart Mobile Robot for Warehouse Managing

Designed a smart mobile robot with mecanum drive and a 4-DOF manipulator with suction gripper for autonomous pick and-place. Implemented in ROS 2 with SLAM Toolbox for mapping, Navigation2 for path planning, sensor fusion and a web-based teleoperation interface with live video.

Line and Wall Following Mobile Platform with Vision Based Sign Detection

Designed and programmed a miniature autonomous taxi capable of line following, wall following, and remote-controlled navigation using a Raspberry Pi. Implemented dynamic mode switching via a mobile app, enabling seamless transitions between manual control, line following, and wall following. Integrated real-time road sign detection across all modes using a hybrid approach: Random Forest classifier for symbolic signs and OCR for speed limit signs, achieving ~99% detection accuracy while maintaining CPU-only compatibility.

Environment Monitoring Station

Developed a multi-sensor IoT station measuring temperature, humidity, light, solar irradiance, wind speed and direction (ultrasonic anemometer), and rainfall (tipping-bucket rain gauge). Integrated calibrated signal conditioning with real-time mobile and web dashboards for continuous environmental monitoring.

Smart Parking Lot

Built an automated parking system for constrained lots using computer vision–based license plate recognition and RFID access control integrated with a cloud-backed database for gate management. Implemented safety sensors and a mobile app for real-time occupancy tracking and user access.

Trimming Sorting Packing Automation Project (Industrial Project)

Collaborated on an industrial automation project to automate the post-processing line of a glove manufacturing plant, focusing on trimming, sorting, and packing stages to enhance efficiency and consistency.
